# 仿饿了么App

# **一、项目介绍**

> 注意：图片、文字等等，这些数据都不是放在数据库里的，都是整理到了本地的一个 json 文件
>
> 中，相当于模拟后台去获取数据。

## **项目技术栈选型**

### **前端技术栈**: vue 3.x、Webpack 4.x Mint-UI组件、axios、vue-router 4.x 、vuex 4.x、Bettle-Scroll、Font Awesome Css框架、Animate Css动画

### **布局**: flexible、flex

### **服务端技术栈：** node+本地 json
